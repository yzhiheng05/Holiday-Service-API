# Holiday Service System (微服务架构节假日查询系统)

### 项目背景
本项目是一个基于微服务架构设计的节假日查询系统，采用网关层、逻辑层、数据库层三层解耦设计。该项目在课程评估中表现优异，展示了多服务联调与数据库持久化能力。

### 核心技术点
* **微服务架构**：通过 Gateway 转发请求，Logic 处理业务逻辑，DB 负责数据持久化。
* **多级缓存策略**：实现 1 小时失效机制，平衡数据实时性与响应速度，减少外部 API 压力。
* **ORM 数据库操作**：使用 SQLAlchemy 实现数据的合并写入 (Merge) 与多维度模糊查询。

### 部署环境
* **云端部署**：系统模块已部署于 PythonAnywhere，保障接口 100% 可访问性。
